# Lista-de-Mascotas
Prueba de sitio de Lista de Mascotas creada con Bootstrap 5 - jquery - Ajax - MySql - PHP - JS - CSS 
Tiene la posibilidad de :  Agregar, editar, borrar filas de la lista de mascotas
